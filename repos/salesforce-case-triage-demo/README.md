# Salesforce Case Triage Demo

Interactive demo for **AI-Driven Case Triage & Resolution** (Agentforce, Data Cloud, Einstein Trust Layer).

[![View live demo](https://img.shields.io/badge/View_Live_Demo-2563eb?style=for-the-badge&logo=google-chrome&logoColor=white)](https://parvez-shaik-portfolio.vercel.app/demos/case-triage)

> Update the demo URL above after you deploy your portfolio to Vercel.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Stack

- React 19 + Vite
- Tailwind CSS
- Simulated case classification and PII masking
