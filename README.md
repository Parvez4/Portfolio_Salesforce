# Parvez Shaik — Portfolio

A minimalist, responsive portfolio site for a Salesforce Developer, built with **Next.js 15**, **React 19**, **Tailwind CSS v4**, and **Framer Motion**.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Deploy

Deploy easily to [Vercel](https://vercel.com) by connecting this repository. No environment variables are required for the default setup.

## Customize content

Edit `src/data/portfolio.ts` to update experience, projects, skills, contact links, and copy.

## Contact form

The contact form opens the visitor’s email client with a pre-filled message to your address. For server-side delivery, connect a service such as Resend, Formspree, or a Next.js API route.
